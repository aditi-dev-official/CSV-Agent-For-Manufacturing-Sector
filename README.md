# 🤖 CSV Agent for Manufacturing Sector (Laptop Dataset)

An AI-powered **CSV Agent** that enables natural language interaction with manufacturing data, allowing users to perform analysis, generate insights, build predictions, and visualize trends effortlessly.

---

## 📌 Project Overview

This project demonstrates how AI can transform traditional data analysis by enabling users to query a CSV file using natural language.

The dataset used (`laptop.csv`) represents manufacturing/product data in the laptop segment.

---

## 🚀 Features

* 💬 Natural Language Querying
* 📊 Automated Data Analysis
* 📈 Data Visualization
* 🤖 Predictive Modeling
* 💡 Insight Generation

---

## 📂 Dataset Details

The dataset includes features like:

* Laptop Brand
* Price
* Specifications (RAM, Storage, etc.)
* Performance-related attributes
* Other manufacturing/product attributes

---

## 🧠 Example Queries

You can ask the agent:

* What are the most expensive laptops?
* Which brand has the highest average price?
* Show correlation between RAM and price

* Identify budget vs premium segments

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/csv-agent-manufacturing.git
```

2. Navigate to project folder:

```
cd csv-agent-manufacturing
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run Jupyter Notebook:

```
jupyter notebook
```
